Goal
Run 5K
Phase 1: 25 min non-stop → Phase 2: build to 5K
Week 1 of 16
Phase 1 — foundation (weeks 1–8)
Wk 1–2 Run 1 min → Walk 2 min × 7 20–25 min
Wk 3–4 Run 2 min → Walk 1 min × 7 25 min
Wk 5–6 Run 5 min → Walk 1 min × 5 25–30 min
Wk 7–8 Run non-stop 25 min
Phase 2 — build to 5K (weeks 9–16)
Wk 9–11 3 easy runs, 25–30 min each easy
Wk 12–13 2 easy + 1 push (3.5–4km) moderate
Wk 14–15 2 easy + 1 push (4–4.5km) moderate
Wk 16 Easy 20 min → Race day 🏁 5K!
Key rules
• Easy pace always — hold a conversation
• Stretch calves + hip flexors after every run
• Get proper running shoes (go to a running store)
• Never run day after heavy leg work
• Target: 32–38 min first 5K finish time
